# Nuvante Clearing Platform - Mockup Design Notes (v0.2)

## 基础信息
- 产品：Nuvante Clearing Platform
- 原型目标：桌面优先的交互验证（P0/P1 页面）
- 当前方向：对齐客户偏好截图的「清爽运营看板」风格

## 风格提取（来自客户参考图）
- 大面积浅灰背景 + 白色卡片容器（弱阴影、细描边）
- 高可读数字指标卡（大字号、低装饰）
- 状态表达使用低饱和标签色（黄=处理中、绿=完成、红=失败）
- 信息区块采用“左主右辅”布局：主交易流 + 账户/告警侧栏
- 时间线、表格、审计日志采用结构化信息层级，强调可扫描性

## 覆盖页面
- `/member`：成员看板 + 发起指令 + 交易列表 + 取消动作
- `/member/transactions/:id`：详情页 + 时间线 + 取消边界
- `/issuer`：发行方供应量/储备 + 交易活动 + Model 2 待处理动作
- `/admin`：全局指标 + 队列表格 + Pause/控制动作 + 趋势图
- `/admin/audit-log`：审计日志过滤与检索

## 交互重点
- 新指令提交：字段校验、成功反馈、Instruction ID 生成
- 取消规则：Settlement 边界前可取消，后续禁用
- 管理动作：支持批量 Pause（示意）并写入审计日志
- 审计过滤：支持 Actor / Event / Reference 组合过滤

## 下一轮建议
- 根据客户确认继续微调字体尺寸与留白密度
- 统一全站英文术语和状态命名（与业务字典一致）
- 若此版通过，可进入 `create-ui-spec` 沉淀设计系统与组件规范
